# Summoners War World of Warcraft (Unity Prototype)

## Opis projektu
Prototyp mobilnej turówki RPG inspirowanej Summoners War: Sky Arena w klimacie World of Warcraft.

## Wymagania
*   Unity Editor (wersja zalecana: 2022.3 LTS lub nowsza)
*   Visual Studio / VS Code dla edycji kodu C#

## Uruchomienie
1.  Otwórz folder projektu w Unity Hub.
2.  Uruchom scenę główną (jeszcze nie utworzona).
3.  Otwórz konsolę (Console Window) aby widzieć logi systemowe.

## Struktura folderów
*   `Assets/Scripts/Models/`: Klasy danych i logiki (Unit, Spell, Synergy).
*   `Assets/Scripts/Engine/`: Silnik walki (Combat Engine).
